# Name: Question Randomizer Web App

A web application built with ASP.NET Core MVC and Visual Studio 2022.

## Features

- Upload DOCX or PDF files
- Extract questions and answers
- Randomize questions
- Answer questions one-by-one
- View user answers vs correct answers
- Restart quiz with new random questions

## Technologies

- ASP.NET Core MVC
- C#
- Bootstrap
- OpenXML
- PdfPig

## How To Run

1. Open solution in Visual Studio 2022
2. Restore NuGet packages
3. Press CTRL + F5
